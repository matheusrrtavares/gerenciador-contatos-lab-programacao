# Gerenciador de Contatos Simples em C 

Este projeto consiste em um sistema de gerenciamento de contatos desenvolvido em modo texto para a disciplina de Laboratório de Programação (Ciclo 1) da UniEvangélica. O objetivo é consolidar fundamentos da linguagem C, como estruturas, ponteiros, strings e recursividade.

## Como Executar o Projeto
## Opção 1:
Pelo Dev-C++ (Interface Gráfica)Abra o arquivo main.c no seu Dev-C++.
Vá no menu Executar e selecione Compilar e Correr (ou aperte a tecla F11).
O terminal abrirá automaticamente com o menu do Gerenciador.
